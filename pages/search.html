does this work?

Type a name into the textbox below. If that name exists in our premium name database, then that name and its assessment will be displayed in the panel to the right.

Or, let's say you just want to know about the traits of people whose name start with a particular letter. Type that letter in by itself, and the general advice will be displayed.

If your name is written with a "-", like Mary-Kate, please write it with a space instead, like Mary Kate.
